#### กรณีเป็น Admin

**การลงทะเบียน**
- ทำขั้นตอนที่ 1-5 ตาม กรณีเป็นผู้ใช้ ข้างบน
- การขอเป็น Admin (ต้องได้รับการอนุมัติจาก Developer หรือ Adminเดิม)
    - คลิกที่ ขีด 3 ขีดด้านซ้ายบน และกดเลือก **ผู้ใช้**
    - เลือกรายชื่อที่ต้องการ
    - ติ๊กถูก ที่ช่อง "ผู้ดูแลระบบ" และกด save

**การอนุมัติขอลงทะเบียนของผู้ใช้**
